# Public Boundary

This repository is a public research and education artifact.

It contains high-level framework notes, conceptual mappings, and toy/spec examples intended to make the Eudaimonic Alignment Framework easier to inspect and discuss.

## Included

- Public research notes
- Educational framework material
- Conceptual mappings between ethical traditions and governance constraints
- Toy/spec examples that do not describe private production systems
- Limitations and disclosure boundaries

## Excluded

This repository does not disclose:

- proprietary system architecture
- private implementation details
- internal audit artifacts
- production logs
- private prompts
- private schemas
- deployment receipts
- customer, user, or partner data
- non-public governance mechanisms
- credentials, keys, or operational secrets

## Safe Use

The material here can be cited as public research framing. It should not be cited as evidence that any deployed AI system is safe, aligned, sentient, autonomous, production-ready, or formally verified.

Any implementation of these ideas would require separate empirical evaluation, security review, governance review, and deployment-specific validation.
